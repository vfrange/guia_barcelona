# Barna · Guia Pessoal

Um guia interativo e pessoal de Barcelona, feito para um casal e seus dois filhos pequenos (1 e 3 anos).

## ✨ Como usar

Basta abrir o arquivo `index.html` no navegador. Não precisa de servidor, não precisa de build, não precisa de nada instalado.

Para publicar no GitHub Pages:

1. Crie um repositório no GitHub
2. Faça upload do `index.html` (e deste `README.md`)
3. Vá em **Settings → Pages**
4. Em "Source", escolha branch `main` (ou `master`) e pasta `/ (root)`
5. Salve e espere 1–2 minutos
6. Seu guia estará em `https://SEU-USUARIO.github.io/NOME-DO-REPO`

## 🎨 O que tem dentro

- **33 lugares** em Barcelona, categorizados por ocasião
- **Seção "Com as crianças"** com lugares testados pra funcionar com bebê e criança pequena
- **Achados escondidos** — lugares pouco conhecidos pela maioria dos turistas
- **Seção "A dois"** — pra quando conseguirem babá
- **Mapa interativo** com todos os pontos
- **Favoritos** — salvos localmente no navegador
- **Filtros** por categoria, custo, compatibilidade com crianças
- **Dicas práticas** atualizadas pra 2026 (horários, reservas, preços)

## 🛠 Tecnologia

100% frontend, single-file HTML. React via CDN, Leaflet pro mapa, Google Fonts pras tipografias. Zero dependência pra instalar.

## 📝 Atualizações

O guia foi construído em 2026, com informações verificadas sobre preços, horários, e mudanças que ocorreram desde 2013 (ano em que foi escrito o roteiro original que inspirou este app).
